| Function               | Converts                            |
| ---------------------- | ----------------------------------- |
| `os.ReadFile()`        | File → `[]byte`                     |
| `string()`             | `[]byte` → `string`                 |
| `strconv.ParseFloat()` | `string` → `float64`                |
| `fmt.Sprint()`         | `float64` (or any value) → `string` |
| `[]byte()`             | `string` → `[]byte`                 |
| `os.WriteFile()`       | `[]byte` → File                     |
